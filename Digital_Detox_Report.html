SNS 중독 해결 플랫폼 — 제품 보고서 v2 | 2026.03

1. 문제

한국 MZ세대의 하루 평균 SNS 사용 시간은 3시간 이상. SNS 과다 사용은 우울, 불안, 수면 장애와 높은 상관관계를 보입니다.

기존 솔루션이 실패하는 이유

솔루션 문제점
스크린타임 (iOS/Android) 알림 무시, 비밀번호 우회 → 결국 본인이 풀어버림
차단 앱 답답함, 반발심 → 앱 삭제
의지력 도파민 중독의 신경과학을 무시한 접근

핵심 문제: 혼자서는 안 된다

SNS 중독 해결을 위한 앱을 만들어도, 혼자 쓰는 앱은 아무도 안 쓴다. 사용자가 스스로 앱을 열고, 기록하고, 챌린지를 수행할 동기가 없다.

2. 가설

"SNS 중독은 혼자 이기는 게 아니라, 친구와 함께 이긴다"

SNS의 핵심은 사회적 연결이다. 그렇다면 해결책도 사회적이어야 한다. 혼자 의지력으로 버티는 게 아니라, 친구들과 함께 줄이는 구조가 필요하다.

3. 제품: 함께하는 디톡스

👥

Digital Detox는?

친구 2명 이상과 함께 시작하는 그룹 디톡스 챌린지 앱

핵심 원칙

  • 혼자 시작 불가 — 반드시 친구 2명 이상과 그룹으로 시작
  • 자동 추적 — 수동 기록 없음, 스크린타임 API 연동
  • 사회적 압력 — 내 사용량이 그룹에 공개됨
  • 함께 성공/실패 — 그룹 목표 달성 시 함께 보상

친구들과 "이번 주 SNS 50% 줄이기" 같은 목표를 함께 설정하고 도전합니다.

📊

그룹 대시보드

누가 얼마나 썼는지 실시간으로 보입니다. 내가 1등이면 자랑, 꼴찌면 압박.

🏆

그룹 보상

목표 달성 시 그룹 전체가 보상. 실패하면 벌칙 (ex: 커피 사기).

5. 왜 작동하는가?

심리학적 근거

원리 적용
사회적 책임감 내가 실패하면 친구들도 실패 → 더 강한 동기
또래 압력 내 사용량이 공개됨 → 창피하니까 줄임
손실 회피 벌칙이 있으면 더 열심히 함
공동 목표 혼자보다 팀이 더 오래 지속

실제 사례

  • 다이어트 앱 — 혼자 하면 3주, 친구와 하면 3개월 지속
  • 운동 앱 (Strava) — 친구 기록 보이면 더 뛰게 됨
  • 금연 — 그룹 금연 프로그램이 개인보다 2배 성공률

6. 누가 쓰나요?

👩‍🎓

대학생 그룹

시험 기간 스터디 그룹이 "시험 끝날 때까지 SNS 줄이기" 챌린지.

👨‍💼

직장 동료

팀 단위로 "업무 시간 SNS 제로" 챌린지. 실패하면 커피 쏘기.

👨‍👩‍👧‍👦

가족

저녁 식사 시간 "모두 폰 내려놓기" 챌린지. 부모-자녀 함께.

💪

자기계발 모임

독서 모임, 운동 모임 등에서 "SNS 대신 책/운동" 챌린지.

공통점: 이미 관계가 있는 사람들. 새로운 사람을 매칭하는 게 아니라, 기존 관계를 활용.

7. 기존 솔루션 vs Digital Detox

기존 솔루션 Digital Detox
혼자 의지력으로 버팀 친구 2명 이상과 함께
앱 차단 / 시간 제한 차단 없이 인식 + 사회적 압력
본인이 설정, 본인이 해제 친구들이 서로 견제

언제든 그만둠 → 삭제 그만두면 친구한테 민폐

10. MVP 범위

Phase 1 (4주)

  • 카카오 로그인 + 친구 초대
  • 그룹 생성 (2-6명)
  • 주간 챌린지 설정
  • 스크린타임 스크린샷 업로드 (수동)
  • 그룹 리더보드
  • 챌린지 성공/실패 판정

Phase 2

  • 네이티브 앱 (스크린타임 API 연동)
  • 실시간 알림 ("친구가 인스타 열었어요")
  • 벌칙 정산 시스템
